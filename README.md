# 📖 Project Overview

The **Blood Donation Platform** is a full-stack MERN application that connects blood donors with patients in need of blood. The platform simplifies the blood donation process through donor registration, blood request management, donor searching, funding support, and role-based dashboards.

The system includes three user roles:

* 🩸 Donor
* 🤝 Volunteer
* 🌐 Admin

Each role has specific permissions to ensure secure and efficient management of blood donation activities.
